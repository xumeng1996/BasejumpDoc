DFF¶

bsg_dff¶

  • Overview

    This dff is positive edge triggered,the cell provides three ports(clk_i,data_i,data_o).

  • Interface

      NAME WIDTH DESCRIPTION
    CLOCK clk_i 1 clock port
    INPUT data_i width_p data input port
    OUTPUT data_o width_p data output port

  • Parameter

    width_p input and output data width.
    harden_p harden_p is a parameter that signals to the tools whether it should use a hardened version of the item, rather than a synthesized version. For example, this could apply to SRAMs, to synchronizers, to particular gates that are placed in a certain place on an ASIC, to particular gates that should not be optimized away in FPGA, etc.
    strength_p set drive strength.

  • Application scenarios

    regfile

  • Circuit structure
